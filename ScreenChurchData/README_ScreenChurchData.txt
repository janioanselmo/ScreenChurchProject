ScreenChurchData
================

Esta pasta guarda os dados locais do ScreenChurch.
Coloque seus arquivos aqui e use Arquivo > Atualizar biblioteca no programa.

Pastas principais:
- bibles/: Bíblias em JSON, incluindo o formato do repositório damarals/biblias.
- songs/: letras em TXT ou JSON. Linha em branco = novo slide.
- media/images/: imagens comuns para projeção.
- media/videos/: vídeos comuns para projeção.
- media/backgrounds/images/: imagens de fundo para letras/Bíblia.
- media/backgrounds/videos/: vídeos de fundo para letras/Bíblia.
- themes/: temas JSON.
- services/: cultos salvos.
- config/: layouts e configurações.
- database/: banco SQLite gerado automaticamente.
- backups/: backups ZIP.

Exemplo de letra TXT:
Título: Nome da Música
Artista: Nome do artista
Autor: Nome do autor
